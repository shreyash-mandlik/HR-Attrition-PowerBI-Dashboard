# 👥 HR Attrition Analysis | Power BI

A Power BI dashboard to analyze employee attrition patterns and identify key factors driving turnover.

## 📌 Objective
Help HR teams understand attrition trends by department, age group, job role, and satisfaction levels.

## 🗂️ Files
| File | Description |
|------|-------------|
| `HR_analytics.pbix` | Main Power BI report file |
| `HR_Analytics.csv` | Processed HR dataset |
| `WA_Fn-UseC_-HR-Employee-Attrition.csv` | Raw IBM HR Attrition dataset |
| `BG12.jpg` | Background image used in dashboard |

## 📊 Dashboard Features
- Overall attrition rate KPI
- Attrition by Department, Age, Gender
- Job satisfaction vs attrition correlation
- Monthly income distribution across roles
- Filters by Education, Marital Status, Job Role

## 🛠️ Tools Used
- Power BI Desktop
- DAX (calculated measures)
- IBM HR Analytics Dataset (Kaggle)

## 💡 Key Insights
- Sales department has highest attrition
- Employees aged 25–35 leave most frequently
- Low job satisfaction strongly correlates with attrition
